Тест 5

«Канальный уровень модели OSI»

Для поиска используйте «CTRL+F» ...
Вопрос:

Какая информация о сетевом адаптере выводится на экран при вызове утилиты ipconfig без параметров?

Ответ:

- Шлюз по умолчанию
- Маска подсети
- IP-адрес

Вопрос:

Какой тип кабеля используется в стандарте 1000BASE-CX?

Ответ:

Твинаксиальный кабель

Вопрос:

Параметр MTU (Maximum Transfer Unit) — это максимальный размер блока данных, который может быть передан на транспортном уровне.

Ответ:

Неверно

Вопрос:

Как в ОС Windows 7/10 открыть командную строку?

Ответ:

1. Win+R
2. cmd

Вопрос:

МАС-адрес 0хFF:FF:FF:FF:FF:FF — это ... адрес.

Ответ:

широковещательный

Вопрос:

На рисунке показан захваченный трафик в программе Wireshark. К какому типу относится МАС-адрес отправителя пакета?

Ответ:

Индивидуальный адрес

Вопрос:

Какие методы позволяют предотвратить переполнение приемных буферов коммутаторов или маршрутизаторов?

Ответ:

Методы управления потоком (flow control)

Вопрос:

Какие сетевые устройства работают на канальном уровне модели OSI?

Ответ:

- Коммутаторы
- Точки доступа
- Сетевые адаптеры
